# Koa Vue SSR Blog

>  This Blog built with Vue 2.0, vue-router & vuex  with server-side rendering by koa.

## Build Setup
Requires Node.js 7+
``` bash
# install dependencies
npm install # or yarn

# serve in dev mode, with hot reload at localhost:3030
npm run dev

# build for production
npm run build

# serve in production mode
npm start
```
